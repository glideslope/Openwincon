# Test Setting  

- Use **Big Buck Bunny** animation    
- **Five** bitrate version
  - 50 kbps
  - 400 kbps
  - 800 kbps
  - 1200 kbps
  - 2000 kbps
- **PSNR** can be expressed using **bitrate**  
  - PSNR(r) = 6.4157 * log(rate) + 22.27  
  - derived from Excel program
